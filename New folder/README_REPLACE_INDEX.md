
پروژهٔ Capacitor آماده برای اپ «نگهبان آب» (com.erfanrahimi.waterguardian) — نسخه 1.0.1

نکات فوری و دستورالعمل:
1. فایل index.html واقعی خودت را جایگزین کن:
   مسیر: ./www/index.html
   (فایل فعلی یک placeholder است)

2. این پروژه آماده است برای پوش به GitHub:
   - بوت کردن مخزن جدید در گیت‌هاب
   - پوش کردن کل پوشهٔ این پروژه به شاخهٔ main

3. GitHub Actions:
   - وقتی push کنی، workflow ای که در .github/workflows/build-apk.yml قرار دارد اجرا می‌شود.
   - پس از اتمام، در Actions → job → Artifacts یک فایل APK برای دانلود ظاهر می‌شود.

4. اگر می‌خواهی من index.html واقعی را در پروژه قرار دهم، فایل index.html را اینجا آپلود کن یا به من بگو تا آن را استخراج کنم.

5. اگر می‌خواهی bundle/sign یا نسخه release تولید شود، نیاز به keystore و تنظیمات release در Gradle داریم — در صورت نیاز من کمک می‌کنم.

موفق باشی — وقتی آماده‌ای، فایل zip را دانلود کن و به گیت‌هاب پوش بزن.
